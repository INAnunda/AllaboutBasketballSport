This project about All basketball You know.
โปรเจคนี้เริ่มทำได้สักพักนึงเเล้ว เเต่ก็ยังไม่ได้พัฒนาอย่างเป็นจิงเป็นจัง ต้องการให้เป็นร้านขายของออนไลน์ ที่ขายได้ทุกๆอย่าง จุดเด่นของมันคือ 
1.ร้านค้า รองเท้าที่ดึงเข้ามา จาก Blender เป็นโมเดล 3D ซึ่งสามารถดูได้ 360 องศา 
2.คลังเก็บของ ทั้งหมดที่เรามี ได้เก็บเอามารวมไว้ในนี้หมดเเล้ว
3.Community สำหรับการพูดคุยระหว่าง คนที่มี Inspiration เกี่ยวกับด้านบาสเกตบอล จะเป็นเรื่องเกี่ยวกับลองเท้า หรือเรื่องพูดคุยระหว่างการวิเคราะห์เกม บาสเกตบอล หรือ อื่นๆ ก็ได้เช่นกัน ตามห้องที่ตัวเอง สนใจที่จะพูดคุย
4.การจองสนามบาส ออนไลน์ โดยที่ไม่ต้องติดต่อ ส่วนตัวอะไรเลย คุณเเค่คลิกๆ เเล้วกดยืนยันการชำระเงิน หลังจากนั้นคุณก็เข้าสนามบาสไปเล่นตามที่คุณต้องการในเวลาที่ได้กำหนดไว้ได้เลย โดยการกดยืนยัน ผู้จองจะได้รับอีเมล เด้งไป จากทางร้าน เเละ ทางเพจ ด้วยเช่นกัน
